This website was developed as a learning exercise for the author into Angular, .Net Core, Azure functions and other technologies that were implemented.

Excluding various calculations the core data within the application is freely available from numerous sources on the web and the author makes no guarantees regarding the accuracy of any data within this website.

For those interested in the design, the Trainer Jockey form data is fetched from a C# .Net Core Web-API that was written to read from an Azure database. The code for the API can be found in this github repo
